# Hub Domine · GPSBI

Versão real inicial do Hub Domine, construída a partir da Constitution e das planilhas operacionais.

## Regras implementadas
- Frete TAK contabilizado uma única vez por viagem.
- Quantidade com frete por viagem única.
- Ticket médio calculado sobre frete sem duplicidades.
- Meta mensal de 150, abrangendo as quatro semanas.
- Horários exibidos em HH:mm.
- Painel, Operação Detalhada, Agenda, Mural e Aprovações.
